Браузеры

Mozilla Firefox (оффлайн-установщики на ваш вкус) Известный браузер, работающий на собственном движке (SpiderMonkey/Gecko). Выпущен в 2002 году разработчиком Mozilla и обновляется по сей день. Имеется возможность использования расширений на Manifest V2. Также имеется возможность кастомизации внешнего вида браузера.
Google Chrome (оффлайн-установщик) Самый популярный и удобный в синхронизации аккаунтов браузер на движке Chromium. Выпущен в 2008 году компанией Google. Не рекомендуется его скачивать и устанавливать на свой компьютер, если хотите использовать расширения, несовместимые с Manifest V3.
Chromium

Chromium
(если вы каким-то чёртом не нашли сайт с загрузкой собранного браузера)		 Веб-браузер с открытым исходным кодом и экспериментальными функциями, разрабатываемый The Chromium Authors. Не рекомендуется его скачивать и устанавливать на свой компьютер, если хотите использовать расширения, несовместимые с Manifest V3.
Tor Browser (нужен VPN) Бесплатный браузер на движке от Mozilla Firefox, позволяющий устанавливать анонимное сетевое соединение в Интернете как с обычными сайтами, так и .onion-домены (Интернет не анонимен! - Тыквенная семечка).
Не рекомендуется его использовать для обхода блокировок часто используемых сервисов, такие как YouTube и Discord.

Медиаплееры и аудиоплееры

MPC-HC (clsid2) Простой, открытый, доступный на Windows, интуитивно понятный медиаплеер, без всяких излишеств. Поддерживает все популярные аудио и видеоформаты. Также поддерживает открытие видео по ссылке (например, видео на YouTube с yt-dlp и VC++ 2010 x86). Умеет в HDR с MPCVR/madVR.
VLC Ещё один простой, открытый, интуитивно понятный медиаплеер. Также поддерживает все популярные аудио и видеоформаты. Доступен на Windows, Linux, macOS и на мобильных ОС.
foobar2000 Бесплатный аудиоплеер на Windows и macOS. Поддерживает популярные и не очень аудиоформаты (даже трекерную музыку!). Имеет много плагинов для работы плеера и комфортного прослушивания музыки в различных форматах.

Торрент-клиенты

qBittorrent Бесплатный и открытый торрент-клиент без рекламы и вирусов, в отличие от uTorrent, MediaGet и их подобных.

Программы для записи видео и трансляций

OBS (Open Broadcaster Software) Бесплатная и открытая программа для записи видео и потокового вещания. Поддерживает плагины для стримов и не только.
Streamlabs Desktop Более простая в настройке программа, форк OBS. Доступна на Windows и macOS. Как и OBS, поддерживает плагины.

Архиваторы

7-Zip Бесплатный и открытый архиватор, доступный на Windows (GUI/консоль), macOS (консольная версия) и Linux (консольная версия).
WinRAR/RAR
($$$)		 Условно-бесплатный архиватор на Windows (GUI/консоль), macOS (консольная версия) и Linux (консольная версия). Если вас раздражают всплывающие окна с предложением купить программу (GUI), лучше данный архиватор не ставить на свой компьютер.

Виртуализация

VMware Workstation 17.6.2 (Windows)

VMware Workstation 17.6.2 (Linux)

VMware Fusion (macOS) (ссылка недоступна)		 Платное (ныне бесплатное для персонального использования) и проприетарное ПО виртуализации для Windows и Linux. Работает стабильнее и поддерживает больше гостевых ОС, чем VirtualBox.

VMware Tools (Windows) | VMware Tools (Windows, x86) | VMware Tools (компонент для Linux) | VMware Tools (компонент для Linux, x86)
Oracle VirtualBox Бесплатное ПО виртуализации для Windows, macOS, Linux и Solaris, но менее стабильно, чем VMware Workstation.

Обход блокировок сервисов

zapret (bol-van)

zapret (Flowseal)		 Одна из надёжных и открытых программ для обхода блокировок без использования VPN. Доступна на Windows и Linux (только от bol-van). Не обходит заблокированные не от ркн IP-адреса.
На данный момент не работает с macOS. Если у кого-то zapret с macOS работает, дайте знать.
v2rayN Клиент, который позволяет обходить блокировки сервисов с использованием VLESS-, VMess-, Trojan-, WireGuard- и Socks-конфигураций. Для работы использует тип ядер Xray или sing-box. Имеет присеты для Китая, России и Ирана.
Требуется ваш созданный VPN и 3x-ui/Marzban для создания собственной конфигурации.
Amnezia VPN Открытый клиент для создания собственного сервера с VPN. Доступен на декстопные и мобильные ОС.
